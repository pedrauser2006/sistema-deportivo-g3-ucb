# 🏆 Sistema Deportivo UCB

Sistema web desarrollado para la gestión deportiva de la Universidad Católica Boliviana.

Proyecto correspondiente a la materia Tecnologías Web II.

---

# 🚀 Funcionalidades

## 📅 Reservas deportivas

- Registro de reservas
- Cancelación de reservas
- Validación de colisiones horarias
- Generación automática de comprobantes PDF
- Envío automático de comprobantes por correo (SMTP)

---

## 💳 Gestión de pagos

- Registro de pagos
- Confirmación de pagos
- Control de deportistas morosos
- Exportación PDF
- Exportación Excel

---

## 🧑‍🎓 Gestión de deportistas

- Registro de deportistas
- Edición de información
- Eliminación de deportistas
- Exportación PDF
- Exportación Excel

---

# 🛠️ Tecnologías Utilizadas

## 🔹 Backend

- Node.js
- Express
- PostgreSQL
- Nodemailer
- PDFKit
- ExcelJS

---

## 🔹 Frontend

- React
- TypeScript
- Vite
- SweetAlert2
- React Hot Toast

---

# ⚙️ Instalación del Proyecto

## 🔹 Clonar repositorio

```bash
git clone https://github.com/pedrauser2006/sistema-deportivo-g3-ucb.git
```

---

# 🔹 Backend

```bash
cd backend
npm install
npm run dev
```

Servidor:

```txt
http://localhost:3000
```

---

# 🔹 Frontend

```bash
cd frontend
npm install
npm run dev
```

Aplicación:

```txt
http://localhost:5173
```

---

# 📄 Exportaciones

El sistema permite:

- Exportar reportes PDF
- Exportar reportes Excel
- Descargar comprobantes automáticos

---

# 📧 SMTP

El sistema utiliza Nodemailer con Gmail SMTP para:

- envío automático de comprobantes
- notificaciones de reservas
